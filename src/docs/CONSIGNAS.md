Requisitos mínimos para aprobar (Nota = 4)
1. Mostrar una lista de películas populares en la página de inicio de la
aplicación
2. Agregar una barra de búsqueda que permita a los usuarios buscar
películas por título. Mostrar los resultados de búsqueda en una lista
de películas
3. Al seleccionar una película de la lista, mostrar los detalles de la
película, como el título, la descripción, el año de lanzamiento y el
póster y algún otro dato más
4. Agregar un filtro de búsqueda, por ejemplo por género o
clasificación por edades
Requisitos para aumentar la nota
5. Agregar un botón "Agregar a favoritos" en la página de detalles de la
película e implementar un sistema de almacenamiento para
guardar las películas favoritas de los usuarios (localStorage,
indexedDB)
6. Diseñar una interfaz de usuario fácil de usar mobile first
7. El listado de películas será obtenido por medio de un fetch a la API
